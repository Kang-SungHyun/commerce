# 🛒 자바 콘솔 커머스 프로그램 과제

자바의 기초 문법과 객체지향 개념을 학습하기 위해 제작한 간단한 쇼핑몰 프로그램입니다.

##  구현 내용
1. **화면 반복 및 종료**: `while`문과 `Scanner`를 사용하여 사용자가 `0`을 누르기 전까지 프로그램이 계속 실행.
2. **화면 이동 및 뒤로가기**: 메인 메뉴에서 카테고리를 선택해 들어가고, 상품 상세 정보를 확인 하고 다시 메인 메뉴로 돌아 오도록 구현.
3. **클래스 역할 분담**: 상품/카테고리/고객/시스템으로 클래스를 분리.
4. **(캡슐화)**: 변수를 마음대로 바꾸지 못하도록 필드를 `private`으로 막고, `Getter`와 `Setter` 메서드를 통해서만 값을 읽고 쓸 수 있게

## 클래스별 역할&설명
- **`Main.java`**: 프로그램이 시작되는 곳 , 상품들과 카테고리를 만들어서 시스템에 넘겨주고 실행.
- **`CommerceSystem.java`**: 메뉴판을 화면에 보여주고, 사용자가 입력한 숫자에 따라 화면을 이동.
- **`Category.java`**: 카테고리에 속한 상품들의 목록(`List`)을 저장.
- **`Product.java`**: 상품 이름, 가격, 설명, 재고 수량 데이터를 가지고 있는 클래스.
- **`Customer.java`**: 고객의 이름, 이메일, 등급 데이터를 가지고 있는 클래스.
